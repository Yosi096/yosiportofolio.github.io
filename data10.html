Deskripsi

Saya mendapatkan project freelance untuk membuat sistem sederhana yang dapat menyimpan dan mengolah data terkait perkebunan. Pengerjaan sistem mulai dari recruitment kebutuhan hingga develop memerlukan waktu kurang lebih 1 bulan.
Sistem dibuat dengan menggunakan Vue.js untuk sisi frontend dan Express.js untuk sisi backend serta menggunakan SQL untuk Databasenya.
