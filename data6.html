Aplikasi Sisfopers.

Deskripsi

Saya berkerja sebagai Front-End Developer di PT. Asiatech Inti Raya sejak September 2021 hingga kini. Selama saya bekerja disini saya terlibat dalam 2 proyek.

Saya bergabung dengan proyek pertama di 3 bulan awal saya di kantor. Proyek ini adalah sistem surat menyurat antar petinggi di TNI AU. Proyek ini menggunakan framework Codeigniter, JavaScript, jQuery dan Bootstrap.

Lalu saya dipindahkan untuk melanjutkan pengerjaan sisfopers. Sistem ini merupakan sistem informasi personel TNI AU Indonesia. Sudah lebih dari 4 bulan proyek ini saya yang pegang untuk sisi front-end nya. Proyek ini menggunakan framework dan bahasa yang sama seperti proyek sebelumnya.
Tampilan yang dapat saya tunjukkan hanya pada bagian halaman login karena terkait privasi sistem.
