Aplikasi MyNotes

Deskripsi Aplikasi

Aplikasi MyNotes merupakan aplikasi sederhana yang saya kerjakan diwaktu luang sebagai bahan untuk latihan atau belajar mandiri. Sebenarnya saya mendapatkan ide aplikasi ini karena permasalahan yang saya hadapi ketika kuliah, namun yang saya buat ini masih sangat sederhana dari yang saya inginkan.

Aplikasi ini dibuat menggunakan framework Nuxt js untuk sisi Front-End. Sedangkan untuk sisi Back-End menggunakan framework Express js dengan database mongo. Pengguna MyUKM terbagi menjadi 2 role yaitu : User dan Admin,

Deskripsi Menu

Terdapat beberapa perbedaan menu untuk setiap role. Namun untuk menu Home selalu sama pada setiap role, yang mana pada menu ini menampilkan berita dan pengumuman. Berikut penjelasannya :

  1. Role Admin
    • Pada role ini, admin dapat menghapus dan mengubah role user.
  2. Role User
    • Menu Utama
      • Pada menu utama terdapat menu To-Do, Note dan Schedule. User dapat melakukan aksi CRUD sederhana.
      Kedepannya pada menu ini akan terus saya kembangkan mulai dari dapat upload file baik berupa gambar maupun pdf. Ada notif pengingat dan lain sebagainya.
    • Menu Upload File
      • Pada menu upload file terbagi menjadi dua yaitu upload gambar dan upload file. untuk saat ini menu ini masih dalam tahap pembuatan.
    • Menu Master
      • Pada menu ini terdapat beberapa menu salah satunya adalah tipe catatan. Pada setiap menu dapat melakukan aksi CRUD.
    • Menu Baru
      • Tidak menutup kemungkinan saya akan menambah menu-menu baru apabila terdapat ide.
