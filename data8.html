Aplikasi MyUkm

Deskripsi Aplikasi

Aplikasi MyUKM merupakan project Tugas Akhir, yang dikerjakan secara berkelompok dengan 3 orang yang terbagi menjadi UI/UX Desaign, Front-End dan Back-End. Aplikasi saya buat ulang untuk sisi FE menggunakan VueJs, sedangkan pada sisi BE tetap menggunakan ExpressJs dengan database mongo db. Pengguna MyUKM terbagi menjadi 3 role yaitu :

  1. User
    Pengguna dengan role ini adalah mahasiswa.
  2. Admin
    Pengguna dengan role ini adalah mahasiswa yang menjabat sebagai pengurus UKM.
  3. Superadmin
    Pengguna dengan role ini adalah pihak akademik yang bertanggung jawab terhadap UKM ITERA.

Deskripsi Menu

Terdapat beberapa perbedaan menu untuk setiap role. Namun untuk menu Home selalu sama pada setiap role, yang mana pada menu ini menampilkan berita dan pengumuman. Ada beberapa menu yang saya tambahkan pada project ini dibandingkan project ketika TA kemaren.

Berikut penjelasan singkat beberapa menu :

  1. Role Admin
    2. Terdapat menu untuk mengolah informasi UKM, berita/pengumuman, keanggotaan dan keuangan terkain pemasukan, pengeluaran dan tabungan.
  2. Role Superadmin
    3. Terdapat menu untuk mengolah akses pengguna aplikasi, menerima file dari admin UKM dan untuk mengolah data master seperti prodi.
  3. Role User
    4. Terdapat menu untuk melihat informasi UKM dan mendaftar sebagai anggota UKM.
